# Diseño Original

Este directorio contiene el diseño original/clásico del marketplace CarDealer.

## Características
- Diseño tradicional de compra/venta de vehículos
- Navegación estándar
- Filtros básicos
- Vista de listado clásica

## Desarrollo
```bash
cd original
npm install
npm run dev
```

El servidor de desarrollo se ejecutará en `http://localhost:5174`

## Independencia
Este diseño es **100% independiente**:
- ❌ No comparte código con otros diseños
- ❌ No comparte autenticación
- ❌ No comparte componentes
- ✅ Aplicación React completamente autónoma
