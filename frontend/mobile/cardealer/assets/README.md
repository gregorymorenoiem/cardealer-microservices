# Assets Directory

This directory contains all app assets.

## Structure

- `images/` - Image files (PNG, JPG, WebP)
- `icons/` - Icon files (SVG, PNG)
- `logos/` - Logo variations
- `fonts/` - Custom fonts

## Image Optimization

- Use WebP format for better compression
- Provide 1x, 2x, 3x variants for retina displays
- Optimize images before adding to assets

## Fonts

Download Inter font from:
https://fonts.google.com/specimen/Inter

Required font weights:
- Regular (400)
- Medium (500)
- SemiBold (600)
- Bold (700)
