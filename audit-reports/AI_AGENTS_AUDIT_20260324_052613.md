## BUGS CRÍTICOS (P0 — bloquean producción)

### 1. **SupportAgent Backend Completamente Caído**
- **URL**: `/api/support/message`
- **Error**: `502 Bad Gateway`
- **Impacto**: Chat de soporte no funciona para usuarios
- **Evidencia**: Error confirmado en DevTools Network

### 2. **WhatsApp Webhook Completamente Roto**
- **URL**: `/api/whatsapp/webhook`
- **Error**: `ERR_INVALID_RESPONSE` + `Failed to fetch`
- **Impacto**: Integración WhatsApp no funcional
- **Evidencia**: Tanto GET verification como POST message fallan

### 3. **LLM Gateway Cost Endpoint Missing**
- **URL**: `/api/admin/llm-gateway/cost`
- **Error**: `404 Not Found`
- **Impacto**: Sin monitoreo de costos LLM
- **Evidencia**: Dashboard admin muestra "Sin datos de costos LLM"

### 4. **PromptCache Metrics No Implementado**
- **URL**: `/api/chat/metrics/prompt-cache`
- **Error**: `404 Not Found`
- **Impacto**: Sin visibilidad de savings de cache
- **Evidencia**: Endpoint no existe

### 5. **DealerChatAgent Single Vehicle No Accesible**
- **Problema**: URLs de vehículos individuales retornan "Vehículo no encontrado"
- **Impacto**: Chat contextual de vehículos no funciona
- **URLs Probadas**: `/vehiculos/toyota-rav4-2023-limited`

## BUGS ALTOS (P1 — degradan experiencia)

### 2. **LLM Gateway Metrics Endpoint Missing**

- **Impacto**: Sin métricas Prometheus para observabilidad
