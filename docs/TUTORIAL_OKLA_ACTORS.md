## 1. Para Compradores (Buyer)

### 1.1 Buscar Vehículos

```
1. Ir a https://okla.com.do/vehiculos
2. Usar los filtros: marca, modelo, año, precio, tipo
3. Los vehículos con badge ⭐ son "Destacados" (prioridad en resultados)
4. Los vehículos con badge 💎 son "Premium" (posición Top 3)
```

### 1.2 Ver Detalle del Vehículo

```
1. Hacer clic en cualquier vehículo del listado
2. Ver fotos, especificaciones, precio
3. Si el dealer tiene badge "Verificado" / "Verificado Dorado" / "Verificado Premium",
   significa que ha sido verificado por OKLA
```

### 1.3 Contactar Dealer

```
1. En la página del vehículo, usar:
   - Botón "WhatsApp" para contacto directo
   - Botón "Llamar" para teléfono
   - Chat IA (ChatAgent) para consultas 24/7
2. El ChatAgent puede responder preguntas sobre el vehículo,
   agendar citas, y dar valoraciones de precio
```

### 1.4 Usar el Soporte OKLA

```
1. Hacer clic en el ícono de chat (💬) en esquina inferior derecha
2. El agente de soporte responde en español 24/7
3. Puedes preguntar sobre:
   - Precios de vehículos
   - Cómo publicar un vehículo (como vendedor particular)
   - Planes de dealer
   - OKLA Coins
   - Cualquier duda sobre la plataforma
```

## 2. Para Dealers

### 2.1 Registro y Verificación

```
1. Ir a https://okla.com.do/registro-dealer
2. Completar formulario con:
   - Nombre del negocio
   - RNC (Registro Nacional de Contribuyentes)
   - Documentos de verificación
3. OKLA revisará y aprobará la cuenta
4. Al aprobarse, recibes el plan LIBRE (gratis)
```

### 2.2 Publicar un Vehículo

```
1. Ir al Dashboard del Dealer
2. Clic en "Nuevo Vehículo"
3. Subir fotos (según tu plan: 10-40 fotos)
4. Completar marca, modelo, año, precio, condición
5. El vehículo pasa a moderación → se publica al aprobarse
```

### 2.3 Destacar un Vehículo

```
Opción A: Incluido en el plan
  - Plan Visible: 3 destacados/mes gratis
  - Plan Pro: 10 destacados/mes gratis
  - Plan Elite: 25 destacados/mes gratis

Opción B: Comprar con OKLA Coins
  - Costo: 50 coins/día, 250 coins/semana, 600 coins/mes
  - Ir a "Publicidad" → "Listing Destacado"
  - Seleccionar vehículo y duración
```

### 2.4 Comprar Visibilidad

```
1. Ir a Dashboard → "Publicidad"
2. Seleccionar producto del catálogo:
   - Listing Destacado ($0.50/día)
   - Top 3 en búsquedas ($1.50/día)
   - Oferta del Día ($15/día)
   - Banner Homepage ($120/mes)
   - Dealer Showcase ($50/mes)
   - Pack Alertas Email ($35/mes)
   - Paquete Visibilidad Total ($175/mes, ahorro $82)
3. Pagar con tarjeta o OKLA Coins
```

### 2.5 Comprar OKLA Coins

```
1. Ir a Dashboard → "OKLA Coins"
2. Seleccionar paquete:
   - Pack Básico:       2,500 coins →  $25
   - Pack Intermedio:   5,500 coins →  $50 (+10% bonus)
   - Pack Profesional: 12,000 coins → $100 (+20% bonus)
   - Pack Dealer:      32,500 coins → $250 (+30% bonus)
3. Pagar con tarjeta
4. Los coins se acreditan instantáneamente
5. Usar coins para comprar productos publicitarios
```

### 2.6 Cambiar de Plan

```
1. Ir a Dashboard → "Plan y Facturación"
2. Ver tabla comparativa de planes:
   ┌──────────────┬────────┬──────────┬────────┬──────────┐
   │              │ LIBRE  │ VISIBLE  │  PRO   │  ELITE   │
   ├──────────────┼────────┼──────────┼────────┼──────────┤
   │ Precio/mes   │  $0    │   $29    │  $89   │  $199    │
   │ Fotos/vehíc. │  10    │   20     │   30   │  40+video│
   │ Destacados   │  —     │  3/mes   │ 10/mes │  25/mes  │
   │ OKLA Coins   │  —     │  $15     │  $45   │  $120    │
   │ Badge        │  —     │ Verific. │ Dorado │ Premium  │
   │ ChatAgent    │  —     │   —      │ 500/mes│ Ilimitado│
   │ Analytics    │  —     │ Básico   │ Avanz. │ Completo │

### 2.7 ChatAgent (IA 24/7)

```
Disponible en planes Pro ($89) y Elite ($199):
- ChatAgent Web: Atiende compradores en tu página
- ChatAgent WhatsApp: Atiende vía WhatsApp
- Pro: 500 conversaciones/mes
- Elite: Ilimitado

## 3. Para Administradores

### 3.1 Dashboard Administrativo

```
1. Login con cuenta admin: admin@okla.local
2. Ir a https://okla.com.do/admin
3. Ver métricas:
   - Total de vehículos publicados
   - Total de dealers
   - Ingresos por planes
   - Ingresos por publicidad
```

### 3.2 Moderación de Vehículos

```
1. Admin → "Moderación"
2. Revisar vehículos pendientes
3. Aprobar o rechazar publicaciones
4. Vehículos aprobados pasan a estado "Active"
```

### 3.3 Gestión de Dealers

```
1. Admin → "Dealers"
2. Ver lista de dealers con estado
3. Aprobar nuevos dealers
4. Suspender dealers por violaciones
5. Ver plan actual de cada dealer
```

### 3.4 Gestión del Homepage

```
1. Admin → "Homepage"
2. Editar secciones:
   - Carousel principal
   - Vehículos Destacados ⭐
   - Vehículos Premium 💎
   - Secciones por tipo (Sedanes, SUVs, etc.)
3. Agregar/quitar vehículos de cada sección
4. Configurar marcas promocionadas
```

### 3.5 Reportes de Publicidad

```
1. Admin → "Publicidad" → "Reportes"
2. Ver campañas activas por dealer
3. Ingresos por producto publicitario
4. Métricas: impresiones, clics, CTR
```

### 3.6 Acreditar OKLA Coins Mensuales

```
Para dealers con plan Visible/Pro/Elite, acreditar coins mensuales:
- POST /api/okla-coins/admin/plan-credit
- Body: { "dealerId": "...", "coinsAmount": 1500 }
  (1500 coins = $15 para plan Visible)
```

## 5. OKLA Coins — Sistema de Créditos

### ¿Qué son los OKLA Coins?

Los OKLA Coins son créditos prepago que puedes usar para comprar productos publicitarios.
**1 OKLA Coin = $0.01 USD**

### Paquetes Disponibles

| Paquete         | Coins Base | Bonus         | Total  | Precio |
| --------------- | ---------- | ------------- | ------ | ------ |
| **Básico**      | 2,500      | —             | 2,500  | $25    |
| **Intermedio**  | 5,000      | +500 (+10%)   | 5,500  | $50    |
| **Profesional** | 10,000     | +2,000 (+20%) | 12,000 | $100   |
| **Dealer**      | 25,000     | +7,500 (+30%) | 32,500 | $250   |

### ¿Cómo obtener OKLA Coins?

1. **Compra directa**: Seleccionar un paquete y pagar
2. **Incluidos en el plan**: Los planes Visible ($15/mes), Pro ($45/mes), y Elite ($120/mes) incluyen coins mensuales
3. **Promociones**: Ofertas especiales y bonus por temporada

### ¿En qué gastar OKLA Coins?

- Listing Destacado: 50 coins/día
- Top 3 en búsquedas: 150 coins/día
- Oferta del Día: 1,500 coins/día
- Banner Homepage: 12,000 coins/mes
- Dealer Showcase: 5,000 coins/mes
- Pack Alertas Email: 3,500 coins/mes
- Paquete Visibilidad Total: 17,500 coins/mes

### 6.6 Pack Alertas Email

- **OKLA Coins**: 3,500/mes

### 6.7 Paquete Visibilidad Total (Bundle)

## API Endpoints de Referencia

### Planes

```
GET  /api/dealer-billing/plans              # Ver todos los planes (público)
GET  /api/dealer-billing/subscription       # Ver mi suscripción actual
GET  /api/dealer-billing/dashboard/{id}     # Dashboard completo
```

### OKLA Coins

```
GET  /api/okla-coins/packages               # Ver paquetes disponibles (público)
GET  /api/okla-coins/wallet                  # Ver mi balance
POST /api/okla-coins/purchase                # Comprar paquete
POST /api/okla-coins/spend                   # Gastar coins
GET  /api/okla-coins/transactions            # Ver historial
```

### Catálogo Publicitario

```
GET  /api/advertising/catalog                # Ver todos los productos (público)
GET  /api/advertising/catalog/{slug}         # Ver producto por slug
GET  /api/advertising/catalog/{slug}/estimate # Estimado de precio
GET  /api/advertising/catalog/category/{cat} # Productos por categoría
```

### Homepage & Secciones

```
GET  /api/homepagesections                   # Ver todas las secciones
POST /api/homepagesections                   # Crear sección (admin)
POST /api/homepagesections/{slug}/vehicles/bulk # Agregar vehículos a sección
```
